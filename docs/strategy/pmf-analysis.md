# PMF（Product-Market Fit）検証テンプレート

> プロダクトが市場に受け入れられているかを定量・定性の両面から検証するためのテンプレート。
> 定期的に（月次/四半期）実施し、PMF達成状況をトラッキングする。

---

## 1. Sean Ellis Test（PMF Survey）

### 質問

> 「もしこのプロダクトが使えなくなったら、どう感じますか？」

### 回答集計テーブル

| 回答選択肢 | 回答数 | 割合 | 目標 |
|-----------|--------|------|------|
| とても残念 | ___ | ___% | **40%以上** |
| やや残念 | ___ | ___% | - |
| あまり残念ではない | ___ | ___% | - |
| もう使っていない | ___ | ___% | - |
| **合計** | ___ | 100% | - |

### 判定基準

| 「とても残念」の割合 | 判定 | アクション |
|-------------------|------|-----------|
| 40%以上 | **PMF達成** | グロース施策に投資可能 |
| 25-39% | PMF接近中 | ヘビーユーザーの特徴を分析し、そのセグメントに集中 |
| 25%未満 | PMF未達 | ピボットまたは大幅な機能改善が必要 |
